# KEY NOTES

A modern notes app

### [See DEMO](https://keynotes.vercel.app)

## Used Tecnologies

- `Nextjs`
- `Firebase`
- `TailwinCSS`
- `React Icons`
- `React Loader Spinner`

## Use Cases

- (C) User can create a note
- (R) User can see the list of created notes
- (U) User can edit a created note
- (D) User can delete created note
